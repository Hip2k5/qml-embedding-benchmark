## Mô tả dự án

Dự án này xây dựng một mô hình Quantum Machine Learning bằng PennyLane để so sánh các phương pháp embedding khác nhau gồm `angle`, `amplitude` và `stateprep` trên bộ dữ liệu giả lập `make_moons`. Chương trình cho phép người dùng chọn phương pháp chạy, huấn luyện mô hình bằng mạch lượng tử tham số hóa, đánh giá độ chính xác trên tập train/test và lưu kết quả ra file CSV.
